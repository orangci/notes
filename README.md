# orangc's notes
My blog and static site generator (written in Python, see [`main.py`](/main.py)), styled with [TailwindCSS](https://tailwindcss.com) & [FlyonUI](https://flyonui.com).

This webpage uses [Umami](https://umami.is) for analytics. These analytics are public [here](https://notes.orangc.net/analytics).
